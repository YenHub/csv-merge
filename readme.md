# CSV Merger

This is a helper util designed to merge CSVs from the command line

## Usage

Create a directory (`./files`) & place the CSVs you intend to merge inside the folder

```bash
# Install Deps
npm i

# Make the director
mkdir -p files

# Run the util
npm start
```
